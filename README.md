# Graficacion-U2

En este documento de GitHub, les presentare el paso a paso de mi proyecto integrador de Graaficacion que consiste en realizar una animacion simple.

## Paso 1

Primero eliminamos el Stroke que Blender nos da por default y lo cambiamos por GPencil, esto lo podemos hacer en el Modo Objeto -> Add -> Grease Pencil -> Blank.

<img width="400" height="400" alt="image" src="https://github.com/24680262/Graficacion-U2/blob/main/STICK1.png"/>

## Paso 2

Creamos el trazo que vamos a ocupar para dibujar en este caso yo use el #402C2OFF.

<img width="400" height="400" alt="image" src="https://github.com/24680262/Graficacion-U2/blob/main/STICK2.png"/>

## Paso 3

Importamos la imagen que vamos a usar de referencia para realizar nuestra animacion.

<img width="400" height="400" alt="image" src="https://github.com/24680262/Graficacion-U2/blob/main/STICK3.png"/>

## Paso 4 

Le damos menos opacidad a la imagen para dibujar a gusto y cada vez que terminemos una imagen saltamos 3 frames en nuestra linea de tiempo.

<img width="400" height="400" alt="image" src="https://github.com/24680262/Graficacion-U2/blob/main/STICK4.png"/>

<img width="400" height="400" alt="image" src="https://github.com/24680262/Graficacion-U2/blob/main/STICK6.png"/>

## Paso 5 

Tomamos todos nuestros dibujos y los encimamos unos sobre otros en orden.

<img width="400" height="400" alt="image" src="https://github.com/24680262/Graficacion-U2/blob/main/STICK5.png"/>

Y listo, ya tienes tu animacion.
